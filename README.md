# GAT - Controle de Corridas

Mini site estático para publicar no GitHub Pages e disponibilizar o APK do app.

## Arquivos principais

- `index.html`: página do site.
- `styles.css`: visual da página.
- `assets/`: imagens usadas no site.
- `GAT-com-recibo.apk`: arquivo baixado pelo botão.

## Publicação no GitHub Pages

Suba todos os arquivos desta pasta para a raiz de um repositório público no GitHub e ative:

- Settings > Pages
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

Depois disso, o site ficará disponível em:

`https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/`
